# Learning Tour

A guided walkthrough of this repository and the intelligent Engineering (iE) approach.

## Who Is This For?

- Developers curious about AI-assisted development
- Teams exploring how to integrate LLMs into their workflow
- Anyone wanting to understand how this repository was built

## How to Take the Tour

**With Claude Code (recommended):**
```bash
claude
/tour
```

**Without Claude Code:**
Read through the stops below in order. Each explains what to look at and why it matters.
